## Key Formulas to Memorize

### Matrices & Determinants
- For multiplication: (m x n)(n x p) = m x p matrix
- Skew-symmetric: a_ij = -a_ji, diagonal = 0
- |adj A| = |A|^(n-1) for n x n matrix
- A(adj A) = |A|.I
- det(kA) = k^n . det(A)
- det(AB) = det(A).det(B)

### Probability
- P(A intersection B) = P(A) x P(B) if independent
- P(at least one) = 1 - P(A')P(B')
- Bayes': P(Ai|E) = P(E|Ai)P(Ai) / Sum of P(E|Aj)P(Aj)

### Vectors
- Dot product: a.b = a1*b1 + a2*b2 + a3*b3
- |a| = sqrt(a1^2 + a2^2 + a3^2)
- Projection of a on b = (a.b)/|b|
- Area of parallelogram = |a x b|

### Application of Derivatives
- Rate of change: dV/dt = (dV/dr)(dr/dt) — chain rule
- f'(x) > 0 means increasing
- Maxima: f'(x) = 0, f''(x) < 0
- Minima: f'(x) = 0, f''(x) > 0

### Integration
- integral of e^x[f(x) + f'(x)]dx = e^x.f(x) + c
- integral 0 to a f(x)dx = integral 0 to a f(a-x)dx

### Differential Equations
- Linear DE: dy/dx + Py = Q
- IF = e^(integral of P dx)
- Solution: y x IF = integral of (Q x IF) dx

### 3D Geometry
- Shortest distance = |(a2-a1).(b1 x b2)| / |b1 x b2|
- Distance from y-axis = sqrt(x^2 + z^2)

### Inverse Trig Ranges
- sin^(-1): [-pi/2, pi/2], domain [-1,1]
- cos^(-1): [0, pi], domain [-1,1]
- tan^(-1): (-pi/2, pi/2), domain R

## Exam Day Strategy

1. **Attempt Section E first** (12 marks in ~20 min). Sub-parts (i) and (ii) are nearly free marks.
2. **Then Section A** (20 marks in ~25 min). Don't spend more than 1.5 min per MCQ.
3. **Section D Q32 (matrices)** is mechanical — do it early while fresh.
4. **Use internal choices strategically:**
   - Q26: Parametric (26B) is usually safer than implicit (26A)
   - Q33: Choose whichever integral you recognize faster
   - Q34: Linear DE (34A) is more formulaic than homogeneous (34B)
5. **Step marks matter.** Even writing the correct formula + identifying variables = 1-2 marks. Never leave a question blank.
6. **Section C Q30 (LPP) and Q31 (Probability)** are the two most predictable 3-mark questions.
